(git init) tạo repo trong local --> (git add .) đưa file vào repo --> (git commit -m "add file index.html") --> sử dụng (git remote add origin +url(copy url khởi tạo repo trên github)) đồng bộ repo trong local với repo trên github --> Tạo một branch bằng (git checkout -b branch_name) để chứa fileindex.html,không đưa trực tiếp file vào main --> (git push -u origin branch_name) push file tới branch vừa khởitạo.

bạn cùng team thực hiện commit,mình sử dụng (git pull) để đồng bộ từ remote về local

conflict khi bạn mình và mình cùng commit -m "resolve conflict" nhưng mình chưa biết.Nếu sau khi mình sửa trên local xong dùng (git add .)--> (git commit -m "resolve conflict)-->(git push) thì error.Lúc này phải dùng (git pull) nhận thông báo CONFLICT contain,lựa chọn commit của bạn hay của mình rồi (git add .)--> (git commit -m "reslove conflict") --> (git push)
